3.1

Introduction to Analytic Geometry

In Chapter 2, we studied vectors and linear mappings. Now we add geometric structure ‚Äî the ability to measure lengths, angles, and distances. This is what transforms abstract linear algebra into a powerful tool for machine learning.

Analytic geometry provides the mathematical framework for understanding similarity between data points. When we ask "how similar are these two documents?" or "how close is this image to the training examples?", we're using concepts from this chapter.

Core Question

Given two vectors x and y, how do we measure:

  • Their individual "sizes" (norms)
  • The angle between them
  • Their distance from each other
  • How much one "points in the direction" of the other (projection)
Agentic AI Connection

Modern AI agents use vector embeddings to represent concepts, tools, and knowledge. When an agent needs to find the right tool for a task, it computes similarities using inner products and distances ‚Äî the exact concepts we'll learn in this chapter.

Cosine Similarity

Cosine similarity is simply the cosine of the angle between vectors:

$$\text{cos\_sim}(x, y) = \frac{\langle x, y \rangle}{\|x\| \cdot \|y\|} \in [-1, 1]$$
Cosine Similarity Interpretation
+1 Vectors point in same direction (identical orientation)
0 Vectors are orthogonal (perpendicular)
-1 Vectors point in opposite directions
Agentic AI Connection

When an AI agent searches for relevant tools or knowledge, it computes cosine similarity between the query embedding and stored embeddings. High similarity (close to 1) indicates high relevance. This is how LangChain and similar frameworks implement semantic search.

Rotations are linear transformations that preserve lengths and angles. They're represented by orthogonal matrices.
